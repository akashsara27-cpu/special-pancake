## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/akashsara27-cpu/special-pancake.git
   cd special-pancake
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
4. Open http://localhost:3000 in your browser

## Customization
Edit config.json to personalize:
- Names (sender, receiver)
- Messages and love letters
- Colors and theme
- Add photos in src/assets/couple_photo/
- Add songs in src/assets/songs/

## Deployment
### GitHub Pages
1. Update package.json with your repository URL
2. Run npm run build
3. Enable GitHub Pages in repository settings
4. Your site will be live at: https://akashsara27-cpu.github.io/special-pancake/
